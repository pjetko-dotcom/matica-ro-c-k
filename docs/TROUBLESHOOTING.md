# Troubleshooting\n\nČaste problémy a ich riešenia.\n\n## GitHub Pages Deployment\n\n### Problem: Vidím iba pozadie, žiadny obsah\n\n**Príčiny:**\n1. GitHub Pages možno stále buduje\n2. Cache v prehliadači\n3. Nesprávny obsah v gh-pages vetve\n4. Wrong `base` path v Vite config\n\n**Riešenia:**\n```bash\n# 1. Čakaj 1-2 minúty\n# GitHub Pages má compilation delay\n\n# 2. Hard refresh\nCtrl+Shift+R (Windows/Linux)\nCmd+Shift+R (Mac)\n\n# 3. Clear cache\n# Settings → Privacy → Clear browsing data\n\n# 4. Skontroluj gh-pages branch\ngit checkout gh-pages\nls -la\n# Má byť: index.html + assets/ zložka\n\n# 5. Skontroluj vite.config.ts\n# base: '/matica-ro-c-k/' must match repo name\n\n# 6. Rebuild a redeploy\nnpm run build\n# Follow DEPLOYMENT.md\n```\n\n### Problem: CSS/JS resources not loading\n\n**Príčiny:**\n- Wrong `base` path\n- Assets not included in dist/\n\n**Riešenia:**\n```bash\n# Check vite.config.ts\ncat vite.config.ts | grep base\n\n# Rebuild\nnpm run build\n\n# Check dist/ folder\nls -la dist/\n# Should have: index.html, assets/\n\n# Redeploy\ngit push origin gh-pages\n```\n\n---\n\n## Development\n\n### Problem: npm install fails\n\n**Riešenia:**\n```bash\n# Clear npm cache\nnpm cache clean --force\n\n# Delete node_modules\nrm -r node_modules\n\n# Reinstall\nnpm install\n```\n\n### Problem: Dev server doesn't start\n\n**Príčiny:**\n- Port 3000 je už obsadený\n- Node version incompatible\n- Dependencies not installed\n\n**Riešenia:**\n```bash\n# Check Node version (must be 18+)\nnode --version\n\n# Check port\nnetstat -an | grep 3000  # Linux/Mac\nnetstat -ano | findstr :3000  # Windows\n\n# Kill process on port 3000\n# Linux/Mac: lsof -ti:3000 | xargs kill\n# Windows: taskkill /PID <PID> /F\n\n# Try different port\nVITE_PORT=3001 npm run dev\n```\n\n### Problem: TypeScript errors in VS Code\n\n**Riešenia:**\n```bash\n# Restart TS Server\n# Ctrl+K Ctrl+F in VS Code\n\n# Check TypeScript version\nnpm list typescript\n\n# Update TypeScript\nnpm install typescript@latest\n```\n\n---\n\n## Data & Storage\n\n### Problem: Data disappeared after refresh\n\n**Príčiny:**\n- localStorage bola vymazan\n- Private browsing mode\n- Storage quota exceeded\n\n**Riešenia:**\n```javascript\n// Check localStorage in console\nlocalStorage.getItem('scout_timetable_nature_v8')\n\n// Check dark mode\nlocalStorage.getItem('scout_dark_mode')\n\n// Restore default data\nlocalStorage.removeItem('scout_timetable_nature_v8')\n// Refresh page - app loads default data\n```\n\n### Problem: Can't delete day/event\n\n**Riešenia:**\n```javascript\n// Clear localStorage completely\nlocalStorage.clear()\n// Refresh page\n\n// Or specific key\nlocalStorage.removeItem('scout_timetable_nature_v8')\nlocalStorage.removeItem('scout_dark_mode')\n```\n\n---\n\n## Dark Mode\n\n### Problem: Dark mode doesn't work\n\n**Riešenia:**\n```javascript\n// Check if enabled\nlocalStorage.getItem('scout_dark_mode')\n\n// Enable dark mode\nlocalStorage.setItem('scout_dark_mode', 'true')\n\n// Check HTML element\ndocument.documentElement.classList.contains('dark')\n\n// Force refresh\nCtrl+F5\n```\n\n---\n\n## Git & GitHub\n\n### Problem: Can't push to GitHub\n\n**Riešenia:**\n```bash\n# Check git config\ngit config user.email\ngit config user.name\n\n# Set if needed\ngit config --global user.email \"your@email.com\"\ngit config --global user.name \"Your Name\"\n\n# Check remote\ngit remote -v\n\n# Try push with verbose\ngit push -v origin main\n```\n\n### Problem: Merge conflicts\n\n**Riešenia:**\n```bash\n# See conflicts\ngit status\n\n# Edit file with <<<< >>>> markers\n# Choose which version to keep\n\n# Mark resolved\ngit add .\n\n# Complete merge\ngit commit -m \"Resolve merge conflicts\"\n```\n\n---\n\n## Performance\n\n### Problem: App is slow\n\n**Riešenia:**\n```javascript\n// Check localStorage size\nObject.keys(localStorage).map(k => k.length).reduce((a,b) => a+b)\n\n// Profile in DevTools\n// F12 → Performance → Record\n\n// Check console for errors\n// F12 → Console\n```\n\n---\n\n## Still Need Help?\n\n1.
